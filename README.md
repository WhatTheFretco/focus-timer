# Focus Timer

A minimal Pomodoro-style focus timer that runs entirely in the browser — no build step, no dependencies.

## Features

- 25-minute focus sessions with 5-minute breaks
- Long break (15 minutes) after every 4 completed sessions
- Animated progress ring and session dots
- Auto-starts the next phase so you never have to babysit it
- Audio chime when a phase ends

## Usage

Open `index.html` in any modern browser and press **Start**.
